# 校园二手物品发布平台 Sprint 计划

## 需求列表

| 模块 | 功能需求 | 需求描述 |
|------|----------|----------|
| 用户管理 | 用户注册 | 用户注册新账户，支持校园邮箱验证 |
| 用户管理 | 登录 | 用户登录系统，支持密码登录 |
| 用户管理 | 个人信息管理 | 用户管理个人信息，包括头像、联系方式等 |
| 物品管理 | 发布物品 | 用户发布二手物品，包括标题、描述、价格、图片等 |
| 物品管理 | 编辑物品 | 用户编辑已发布的物品信息 |
| 物品管理 | 删除物品 | 用户删除已发布的物品 |
| 物品浏览 | 物品列表 | 用户浏览二手物品列表，支持分页和排序 |
| 物品浏览 | 物品详情 | 用户查看物品详细信息 |
| 交易管理 | 询价/议价 | 用户对物品进行询价和议价 |
| 收藏 | 收藏物品 | 用户收藏感兴趣的物品 |
